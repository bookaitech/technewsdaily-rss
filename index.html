How this works

GitHub Pages will serve files from the docs/ folder on the main branch. This repository includes a GitHub Actions workflow that deploys the docs/ folder to Pages automatically on push.

To publish your own site, put static HTML/CSS/JS into this folder, or update the workflow to build from a different source.

Next steps

  • Check the repository Settings → Pages and ensure the source is set to GitHub Actions (the workflow sets the content).
  • Optionally add a CNAME file to docs/ to use a custom domain.
